# ABOUT SUBMISSION
## Intro
The Beetlejuice Cursifier is a fanbase site dedicated to the movie character Beetlejuice by Tim Burton. It features two main pages, the first being a sort of mini game which allows for a user to create an automatically generated (and perhaps also funny) curse.
The 'history' page features a few paragraphs of text detailing the history, both canonical and filmographical, regarding the popular halloween film character.

## Features
### All Pages
Clear theming using colours which mirror the pop-culture icon of Beetlejuice (purple, green, black and white).
#### Navbar
The logo and navigational elements
#### Footer

### 🦇 Game 🦇
The user is invtited to enter a name of the person they wish to receive a Beetlejuice themed curse. The curse is then produced in thre stages, and the user can repeat any or all of those stages to find the most appropriate curse for their victim. They can then click a button to complete the curse, providing some images of the famous character and the final curse. They are then given the option to play again.
### 🦇 History 🦇
The user is able to read a brief history of Beetlejuice in pop-culture and also access the trailers of the original film, the sequel and the cartoon series.
